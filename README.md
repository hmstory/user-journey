# 말해보카 User Journey Map

말해보카와의 사용자 여정을 시각화한 인포그래픽 웹사이트입니다.

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- lucide-react (아이콘)

## 설치 및 실행

### 개발 서버 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요.

### 빌드

```bash
npm run build
```

### 배포

빌드 후 `dist` 폴더의 내용을 GitHub Pages에 배포하세요.

## 주요 기능

- 인터랙티브 감정 곡선 그래프
- SVG 기반 반응형 그래프 (viewBox 0-100 사용)
- 타임라인 카드
- PDF 저장/인쇄 기능
- 반응형 디자인
